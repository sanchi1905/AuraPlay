# ⌨️ Keyboard Shortcuts Reference

## Playback Controls
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Space` | Play/Pause | Toggle playback of current track |
| `→` (Right Arrow) | Next Track | Skip to the next song in playlist |
| `←` (Left Arrow) | Previous Track | Go back to previous song |

## Volume Controls
| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` (Up Arrow) | Volume Up | Increase volume by 5% |
| `↓` (Down Arrow) | Volume Down | Decrease volume by 5% |
| `M` | Mute/Unmute | Toggle audio mute |

## Playback Modes
| Shortcut | Action | Description |
|----------|--------|-------------|
| `S` | Shuffle | Toggle shuffle mode on/off |
| `R` | Repeat | Cycle through repeat modes (None → All → One) |

## Navigation Tips
- Use **Tab** to navigate between interactive elements
- Press **Enter** or **Space** to activate focused buttons
- All controls are fully keyboard accessible
- Focus indicators show which element is currently selected

## Accessibility Notes
- Screen reader compatible with full ARIA labels
- Keyboard navigation follows logical tab order
- Visual focus indicators for all interactive elements
- Respects `prefers-reduced-motion` setting

---

*Tip: Press `Space` to quickly play/pause without using the mouse!*
